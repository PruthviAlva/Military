# Military Asset Management System

A comprehensive full-stack solution for managing military assets (vehicles, weapons, ammunition) across multiple bases with role-based access control, transparency, and accountability.

## Project Structure

- `/backend` - Node.js + Express REST API with PostgreSQL
- `/frontend` - React TypeScript web application
- `/docs` - Project documentation and database schema

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Axios
- **Backend**: Node.js, Express.js, PostgreSQL, JWT, Winston (logging)
- **Database**: PostgreSQL with comprehensive schema for assets, audit trails
- **Deployment**: Vercel (Frontend), Render (Backend)

## Core Features

1. **Dashboard** - Display key metrics and asset movements
2. **Purchases** - Record and track equipment purchases
3. **Transfers** - Manage asset transfers between bases
4. **Assignments & Expenditures** - Assign assets and track usage
5. **RBAC** - Role-based access control (Admin, Base Commander, Logistics Officer)
6. **API Logging** - Comprehensive audit trail for all operations
